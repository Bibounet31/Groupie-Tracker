Groupie Trackers

Artist

Groupie Tracker est une application web en Go qui permet d’afficher, rechercher et filtrer des artistes ainsi que leurs concerts à partir de l’API officielle Groupie Trackers. Le site propose une page d’accueil, une liste d’artistes, une page détaillée pour chaque artiste, une barre de recherche, des filtres, une carte interactive.
